# 简介

### TlUI(tl-element--plus) 是二次封装 Element-Plus 常用业务组件的一个组件库。大幅提升了开发效率。

## 技术栈

Vue3+Vite+TS+Element-plus+Vitepress+FullCalendar

## 组件在线文档

https://tlui.icbox.top

## npm

```
npm install tl-elements-plus --save
```
